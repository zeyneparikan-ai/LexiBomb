# 💣 LexiBomb - Kelime İmha Hattı

LexiBomb, oyuncuların zamana karşı yarışarak harfleri doğru sırayla topladığı ve dinamik bilmeceleri çözmeye çalıştığı, web tabanlı fütüristik bir kelime bulmaca ve arcade oyunudur. Proje; dinamik DOM yönetimi, ses sentezi (Text-to-Speech) ve tarayıcı tabanlı asenkron mekaniklerin entegrasyonu ile geliştirilmiştir.

## 🚀 Öne Çıkan Özellikler
- **🎙️ Yapay Ses Motoru (Text-to-Speech):** Oyundaki tüm ipuçları ve geri bildirimler, tarayıcının yerel ses sentezi motoru kullanılarak oyuncuya sesli bir asistan gibi aktarılır.
- **⚡ Akıcı DOM Yönetimi:** Harflerin havuzdan anlık olarak üretilmesi, doğru/yanlış kontrolleri ve arayüz güncellemeleri saf JavaScript ile dinamik olarak yönetilir.
- **⏳ Zamana Karşı Yarış:** Oyuncu her kelime için kısıtlı bir süreye sahiptir. Doğru hamleler puan kazandırırken, yanlış harf tercihleri veya sürenin bitmesi doğrudan oyunun sonlanmasına (bomba patlamasına) yol açar.
- **🎨 Minimalist Siberpunk Arayüz:** Gözü yormayan, karanlık mod odaklı ve kullanıcı dostu modern web tasarımı.
- **🎉 Çözülenler Arşivi:** Başarıyla tamamlanan kelimeler anlık olarak sağ paneldeki başarı listesine eklenir ve oyuncunun ilerlemesi görsel olarak takip edilebilir.

## 🛠️ Kullanılan Teknolojiler
- **HTML5:** Oyunun semantik iskeleti ve panel yapılarının kurulması için.
- **CSS3:** Modern, temiz ve karanlık tema (Dark Mode) görsel kimliği için.
- **Saf JavaScript (Vanilla JS):** Web Speech API (Sesli okuma motoru), asenkron durum yönetimi, dizi/kelime havuzu manipülasyonları ve olay dinleyicileri için.
